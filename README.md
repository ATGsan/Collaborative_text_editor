# Как собрать проект

1. На вашем компьютере должен быть установлен gRPC и Protobuf;
2. Далее надо выполнить следующие команды:
'''shell
    mkdir -p cmake/build
    cd cmake/build
    cmake ../../service
    make(-j)
'''
3. Появятся файлы client и server которые можно запустить.
